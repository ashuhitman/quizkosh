- CSS
- Rdux Toolkit

### server
- Nodejs
- Mongodb
- jsonwebtoken


## Features

- Authentication
- Quiz Creation, Updation, Deletion

## How to use

1. Clone the repo

```bash
git clone https://github.com/ashuhitman/quizkosh
```

2. Install dependencies

```bash
npm install
```

3. Start the dev server

```bash
npm run dev
```
